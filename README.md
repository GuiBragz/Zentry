# Zentry

O Zentry é um sistema operacional de produtividade minimalista, projetado para oferecer foco total sem fricção. Desenvolvido para rodar integralmente no navegador, o sistema elimina a necessidade de cadastros ou sincronização em nuvem: seus dados são armazenados localmente, garantindo total privacidade e acesso imediato.

## Funcionalidades

*   **Gestão de Tarefas (Z-Cards):** Organização de tarefas e atividades únicas com interface simplificada.
*   **Pomodoro Integrado:** Técnica de foco configurável para manter o ritmo de trabalho.
*   **Ambiente de Concentração:** Player de áudio integrado com curadoria Lofi.
*   **Persistência Local:** Salvamento automático de dados via cookies e LocalStorage.
*   **Design Minimalista:** Interface responsiva com temas Dark e Light.

## Tecnologias

*   **Frontend:** HTML5, CSS3 (Glassmorphism) e JavaScript (Vanilla).
*   **Arquitetura:** Aplicação web estática com foco em performance e ausência de dependências externas.
